Tags em HTML

Agora é a gora de mostrar o que você está aprendendo de HTML5. Esse exerecício faz uma lista de coisas que podem ser adicionadas em uma pagina. O desafio aqui é arrumar uma maneira que, ao passar o mouse sobre o texto, mostre qual é a tag ultilizada para fazer isso. Olha ai um exemplo:

  • Paragrafo
  • Titulos
  • Imagens
  • Abreviações
  • Codigo-fonte
  • Listas numeradas
  • Listas com demarcadores
